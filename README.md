# Focus List

A modern, accessible to-do list application built with semantic HTML5, CSS3, and vanilla JavaScript.

## Features

- Create, read, update, and delete tasks
- Local storage persistence
- Active, completed, and all filters
- Task counter and clear-completed action
- Responsive and accessible interface

## Run locally

Open index.html in a browser, or serve the folder with a static server such as:

```bash
python3 -m http.server 8000
```

Then visit http://localhost:8000.
